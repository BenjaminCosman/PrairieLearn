Let $a, b, c, d, e$ be some integers such that e is odd. Prove that for any integers $m, n$, if $am^2 + bm + cn^2 + dn + e$ is even then either $m$ is odd or $n$ is odd

Proof by Contrapositive: Take some $m \in \mathbb{Z}$ Take some $n \in \mathbb{Z}$ Assume $m$ is even Assume $n$ is even $\exists l \in \mathbb{Z}$ such that $m = 2l$ $\exists k \in \mathbb{Z}$ such that $n = 2k$ $am^2 + bm + cn^2 + dn + e$ $= a(2l)^2 + b(2l) + c(2k)^2 + d(2k) + e$ $= 2(2al^2 + bl + 2ck^2 + dk) + e$ Since $e$ is odd, $2(2al^2 + bl + 2ck^2 + dk) + e$ is odd $QED$
